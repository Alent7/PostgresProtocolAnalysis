# PostgresProtocolAnalysis
PostgreSQL数据库协议解析

这是一个对PostgreSQL数据库进行协议解析的程序

postgresql.pcap为利用Wireshark工具抓包生成的原始数据包文件

程序每次对200条数据包进行分析，按C键继续

分析结果：
